# TaylorE — Разложение e⁻ˣ в ряд Тейлора с визуализацией сходимости

Консольное приложение для Windows, которое вычисляет значения экспоненты e⁻ˣ через разложение в ряд Тейлора и выводит их в форматированную таблицу вместе с порядком скорости сходимости ряда при разных точностях. Позволяет наглядно оценить, как количество членов ряда зависит от требуемой точности и значения аргумента.

---

## 🛠 Технологический стек

При проектировании архитектуры приложения упор делался на скорость обработки запросов и модульность.

*   **Язык разработки:** C++17
*   **Стандартная библиотека:** STL (`iostream`, `cmath`, `iomanip`, `limits`, `locale`)
*   **Платформа:** Windows (WinAPI: `windows.h`, `shlobj.h`, `conio.h`)
*   **Сборка:** CMake 3.16+ (или ручная: MSVC / MinGW-w64)

---

## 🚀 Ключевой функционал

*   **Управление вычислениями:** Вычисление e⁻ˣ через ряд Тейлора с рекуррентным пересчётом члена
*   **Автоматизация логики:** Два порога точности (ε = 0.1 и ε = 0.000001), автоматический подсчёт числа членов ряда, оценка порядка сходимости
*   **Интеграции:** Пакетный расчёт для диапазона (xn, xk, шаг dx), табличный вывод с форматированием
*   **Валидация и безопасность:** Валидация ввода — проверка типов данных и логики диапазона (xn < xk, dx > 0)

### Алгоритм — разложение e⁻ˣ в ряд Тейлора

Ряд сходится на всей числовой оси:

```
e⁻ˣ = 1 − x + x²/2! − x³/3! + … = ∑ (−x)ⁿ / n! , n = 0 … ∞
```

Рекуррентная формула (один член через предыдущий):

```
term₀ = 1
termₖ₊₁ = termₖ · (−x) / (k + 1)
```

## 💻 Локальное развертывание

Для запуска проекта в изолированном окружении вам понадобятся **CMake 3.16+** и компилятор с поддержкой C++17.

### 1. Клонирование репозитория

```bash
git clone https://github.com/<ваш-username>/TaylorE.git
cd TaylorE
```

### 2. Сборка через CMake (рекомендуется)

```bash
cmake -B build -DCMAKE_CXX_STANDARD=17
cmake --build build
```

Готовые исполняемые файлы — в `build/`.

### 3. Ручная сборка (без CMake)

**MSVC:**
```bash
cl /EHsc /std:c++17 main.cpp /Fe:TaylorE.exe
```

**MinGW-w64:**
```bash
g++ main.cpp -o TaylorE.exe -std=c++17 -static
```

### 4. Запуск

```bash
./TaylorE.exe
```

Далее — интерактивный ввод: `xn` (начальное значение x), `xk` (конечное значение x), `dx` (шаг).
