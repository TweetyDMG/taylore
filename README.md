# TaylorE — Разложение e⁻ˣ в ряд Тейлора с визуализацией сходимости

Консольное приложение для Windows, которое вычисляет значения экспоненты e⁻ˣ через разложение в ряд Тейлора и выводит их в форматированную таблицу вместе с порядком скорости сходимости ряда при разных точностях. Позволяет наглядно оценить, как количество членов ряда зависит от требуемой точности и значения аргумента.

## 🚀 Ключевой функционал

- **Вычисление e⁻ˣ через ряд Тейлора** — разложение экспоненты в степенной ряд с рекуррентным пересчётом члена.
- **Два порога точности** — ε = 0.1 (грубая) и ε = 0.000001 (высокая) для демонстрации разницы в скорости сходимости.
- **Автоматический подсчёт числа членов ряда** — сколько итераций потребовалось для достижения заданной ε.
- **Оценка порядка сходимости** — эмпирическая метрика O(log|R| / log N).
- **Пакетный расчёт для диапазона** — пользователь задаёт xn, xk и шаг dx.
- **Табличный вывод с форматированием** — ровные колонки, рамки, выравнивание через `setw` / `setprecision`.
- **Установщик с выбором директории** — `setup.exe` через диалог `SHBrowseForFolderW`.
- **Валидация ввода** — проверка типов данных и логики диапазона (xn < xk, dx > 0).

### 🗄️ Алгоритм — разложение e⁻ˣ в ряд Тейлора

Ряд сходится на всей числовой оси:

```
e⁻ˣ = 1 − x + x²/2! − x³/3! + … = ∑ (−x)ⁿ / n! , n = 0 … ∞
```

Рекуррентная формула (один член через предыдущий):

```
term₀ = 1
termₖ₊₁ = termₖ · (−x) / (k + 1)
```

При больших |x| требуется больше членов — программа наглядно это демонстрирует через колонку `numTerms`.

## 💻 Локальное развёртывание

### 1. Клонирование

```bash
git clone https://github.com/<username>/TaylorE.git
cd TaylorE
```

### 2. Сборка через CMake (рекомендуется)

```bash
cmake -B build -DCMAKE_CXX_STANDARD=17
cmake --build build
```

Готовые исполняемые файлы — в `build/`.

### 3. Ручная сборка (без CMake)

**MSVC (Visual Studio Developer Command Prompt):**
```bash
cl /EHsc /std:c++17 main.cpp /Fe:TaylorE.exe
cl /EHsc /std:c++17 setup.cpp /Fe:setup.exe
```

**MinGW-w64:**
```bash
g++ main.cpp -o TaylorE.exe -std=c++17 -static
g++ setup.cpp -o setup.exe -std=c++17 -static -lole32 -lshell32
```

### 4. Запуск

```bash
./TaylorE.exe
```

Далее — интерактивный ввод:
- `xn` — начальное значение x
- `xk` — конечное значение x
- `dx` — шаг

```
Введите начальное значение x (xn): 0
Введите конечное значение x (xk): 5
Введите шаг изменения x (dx): 1
```

### 5. Сборка дистрибутива (опционально)

```bash
cmake --build build --config Release
7z a TaylorE-1.0.0.7z build/Release/TaylorE.exe build/Release/setup.exe README.md
```
